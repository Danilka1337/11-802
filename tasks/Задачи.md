## HomeWork2

Пусть задана последовательность целых чисел 
A = {a1, ... aN}, где aN = -1, N -> inf

Вывести произведение чисел, сумма цифр которых нечетна.

71 -> 7 + 1 = 8, четное
23 -> 2 + 3 = 5, нечетное, кидаем в произвдение
86 -> 8 + 6 = 14, четное 
17 -> 1 + 7 = 8, не интересно

Вывод -> 23
